## 프로젝트 소개 
- Student Mangement System
  
- 학생 관리 시스템으로 학생의 정보를 입력 받고 저장, 검색, 정렬하는 기능을 제공합니다.

<br>

## 개발 기간
- 2024.2.7(금) ~ 2024.2.10일(월) : 학생관리시스템 설계구조(draw.io) 작성
  
- 2024.2.11(화) ~ 2024.2.17(월) : 학생관리시스템 코드 구현 (파일쓰기,읽기)
  
- 2024.2.18(화) ~ 2024.3.7(금) : 학생관리시스템 + 인사관리시스템 통합관리 시스템 코드 구현(MySQL DataBase 연동)

<br>

- 학생 정보 정렬
